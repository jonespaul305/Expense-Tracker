💰 Expense Tracker Web Application
A full-stack Expense Tracker web application that allows users to record, manage, and analyze their daily income and expenses. The application provides a clean dashboard to track financial activity and maintain better control over personal finances.
This project is built using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript.

📌 Features
User-friendly interface for tracking expenses
View all transactions in a dashboard
Edit and delete transactions
Categorize expenses
Real-time data updates
MySQL database storage
Responsive UI design
Secure backend API
